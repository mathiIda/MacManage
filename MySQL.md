# MySQL

## 1.备忘

- 可以通过Settings管理MySQL
- MySQL位置：usr/local/mysql（mysql-8.0.28-macos11-arm64）
- 账密：root/niuNIU123...
- 环境变量(已添加进.zshrc)：MYSQL_HOME=/usr/local/mysql;PATH=$MYSQL_HOME/bin:$PATH;PATH=$MYSQL_HOME/support-files:$PATH

## 2.常用命令

- 查看版本

  ```shell
  mysql --version
  ```
